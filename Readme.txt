
欢迎使用“小嵌”操作系统SmallRTOS；

SmallRTOS的目录结构如下：

Kernel： 存放SmallRTOS的内核文件OS及和CPU相关的移植文件；
Demo：   存放SmallRTOS提供的示例文件；
Doc：    存放SmallRTOS相关说明/教程文档；
License：存放SmallRTOS使用许可；

如有疑问，请访问官方网站进行咨询：
